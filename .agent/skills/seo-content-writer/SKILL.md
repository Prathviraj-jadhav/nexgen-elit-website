# SEO Content Writer

## When to Use This Skill

- Writing blog posts targeting specific keywords
- Creating landing pages optimized for search
- Developing pillar content or topic clusters
- Writing product/service descriptions
- Creating how-to guides and tutorials
- Writing comparison and review articles
- Refreshing existing content with SEO improvements

---

## Step 1: Gather Requirements

Before writing, confirm:

```markdown
### Content Brief

**Primary Keyword**: [main keyword to rank for]
**Secondary Keywords**: [2-5 related/supporting keywords]
**Content Type**: [blog post / guide / landing page / comparison]
**Target Word Count**: [1000 / 1500 / 2000 / 3000+]
**Target Audience**: [who reads this and what's their expertise level]
**Search Intent**: [informational / commercial / transactional / navigational]
**Tone**: [professional / conversational / technical / friendly]
**CTA Goal**: [email signup / product purchase / contact form / download]
**Competitor URLs**: [top 3 ranking pages to beat  optional]
```

## Step 2: CORE-EEAT Pre-Write Checklist

Apply these quality standards before writing:

| Code | Standard | Requirement |
|------|----------|-------------|
| C01 | Intent Alignment | Title promise matches content delivery |
| C02 | Direct Answer | Core answer within first 150 words |
| C06 | Audience Targeting | State who this article is for |
| C10 | Semantic Closure | Conclusion answers opening question + next steps |
| O01 | Heading Hierarchy | H1→H2→H3, no level skipping |
| O02 | Summary Box | Include TL;DR or Key Takeaways |
| O06 | Section Chunking | Each section = single topic; paragraphs 3–5 sentences |
| O09 | Information Density | No filler; consistent terminology |
| R01 | Data Precision | ≥5 precise numbers with units and sources |
| R02 | Citation Density | ≥1 external citation per 500 words |
| R04 | Evidence-Claim Mapping | Every major claim backed by evidence |

| O08 | Anchor Navigation | Table of contents with jump links for 2000+ word content |

## Step 3: SEO Research & Planning

### SERP Analysis
Before writing, analyze what's ranking:
- What format do top results use? (listicle, guide, comparison?)
- What's the average word count?
- What H2 sections appear frequently?
- What "People Also Ask" questions appear?
- Are there featured snippet opportunities?

### Keyword Map
```markdown
Primary keyword:    [keyword]  Use in: title, H1, first 100 words, conclusion, URL
Secondary keywords: [list]    Use in: H2s, body paragraphs (natural placement)
LSI/Related terms:  [list]   Sprinkle naturally throughout
PAA Questions:      [list]   Use as H2/H3s or in FAQ section
Long-tail variants: [list]   Answer within relevant sections
```

## Step 4: Title & Meta Description

### Title Requirements
- Include primary keyword (ideally at the start)
- Under 60 characters for full SERP display
- Power words: Complete, Ultimate, Guide, Beginner, Free, Fast, Easy, How to
- Match search intent

**Generate 3 title options:**
```
1. [Primary Keyword]: The Complete [Year] Guide  (XX chars)
2. How to [Action] with [Primary Keyword]  [Benefit]  (XX chars)  
3. [Number] Best Ways to [Goal with Primary Keyword]  (XX chars)

Recommended: [option] because [reason]
```

### Meta Description
- 150–160 characters
- Include primary keyword naturally
- Include clear value proposition
- End with actionable hook

## Step 5: Content Structure

```markdown
# [H1  Primary Keyword Here]

**TL;DR / Key Takeaways** (for 1500+ word articles):
- [Point 1]
- [Point 2]
- [Point 3]

## Table of Contents (for 2000+ words)
- [Section 1](#anchor)
- [Section 2](#anchor)

---

[INTRO  100-150 words]
- Hook sentence that grabs attention
- Problem statement (why this matters to reader)
- Promise (exactly what they'll learn)
- Primary keyword in first 100 words
- Brief overview of what's covered

## [H2  Secondary Keyword or Common Question]

[Section intro  1-2 sentences]

[Main content with evidence and examples]

> **Pro Tip**: [Highlighted insight that adds real value]

| Comparison | Option A | Option B |
|------------|----------|----------|
| Feature X  | ✅ Yes   | ❌ No    |

### [H3  Sub-topic]

[Detailed content]

- Bullet point 1
- Bullet point 2
- Bullet point 3

## [H2  Next Major Section]

...continue with all major sections...

## Frequently Asked Questions

### [Question from PAA or common search query]?

[Direct answer in 40-60 words  formatted for featured snippet opportunity]

### [Question 2]?

[Answer]

## Conclusion

[Summary of 3-5 key points  include primary keyword]

[Final insight or call to action]

**[CTA]: [Clear next step for the reader]**
```

## Step 6: On-Page SEO Checklist

```markdown
### Keyword Placement
- [ ] Primary keyword in H1 (only one H1)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Primary keyword in conclusion/last paragraph
- [ ] Primary keyword in meta description
- [ ] Secondary keywords in H2s/H3s (natural placement)
- [ ] Related terms throughout body (LSI)

### Content Quality
- [ ] Comprehensive coverage  addresses search intent fully
- [ ] Original insight, data, or perspective
- [ ] Actionable takeaways readers can implement
- [ ] Real examples, statistics, or case studies
- [ ] Citations to authoritative external sources (2-3 minimum)

### Readability
- [ ] Paragraphs 3-5 sentences max
- [ ] Varied sentence length (mix short and medium)
- [ ] Bullet points for lists of 3+ items
- [ ] Bold for key terms (but not overused)
- [ ] Table of contents for 2000+ word content

### Technical
- [ ] Internal links to 2-5 relevant pages
- [ ] External links to authoritative sources
- [ ] URL slug includes primary keyword
- [ ] Image alt text contains keyword where natural
- [ ] Schema markup recommended (Article, FAQ, HowTo)
```

## Step 7: 2025 SEO Updates (AI Overview / SGE)

For AI Overview (Search Generative Experience) optimization:

```
1. Direct answers early  AI summaries pull from first 150 words
2. Structured data  FAQ and HowTo schema increases AI citation chances
3. Unique data  Stats, studies, and original research get cited more
4. Entity optimization  Use full entity names; build topical authority
5. Conversational queries  Target natural language questions as H2s
6. Credibility signals  Author bio, publication date, organization name
```

## Step 8: Featured Snippet Optimization

```markdown
**Definition snippets**: "[Term] is [clear 40-60 word definition]."

**List snippets**: Use H2 followed by a numbered/bulleted list
  ## How to [Task]
  1. Step one description
  2. Step two description
  3. Step three description

**Table snippets**: Comparison tables with clear headers

**How-to snippets**: 
  ### Step 1: [Action]
  [Description]
  ### Step 2: [Next Action]
  [Description]
```
